# 11 - ReactJS

Hands-on lab exercises covering ReactJS fundamentals as part of the Cognizant GenC Full-Stack training program.

## Prerequisites

- Node.js & NPM ([Download](https://nodejs.org/en/download/))
- Visual Studio Code
- Basic knowledge of HTML, CSS, JavaScript, and ES6

## How to Run Any Exercise

```bash
# Navigate into the exercise folder
cd HOL-XX-folder-name

# Install dependencies
npm install

# Start the development server
npm start

# Open http://localhost:3000 in your browser
```
