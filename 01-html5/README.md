# Module 01 — HTML5

Theme: **Local Community Event Portal**. Ten exercises building the portal's markup foundation —
semantic structure, navigation, forms, media, events, storage, and geolocation.

Each exercise is self-contained. Open its `index.html` in a browser (or use VS Code Live Server).

## Exercises

| # | Folder | Focus |
|---|--------|-------|
| 01 | `ex01-base-template` | HTML5 boilerplate, semantic `<header>/<main>/<footer>`, comments |
| 02 | `ex02-navigation-linking` | `<nav>`, in-page anchors, external link with `target="_blank"` |
| 03 | `ex03-welcome-banner` | `id` vs `class`, inline + internal CSS, `<div>`/`<span>` |
| 04 | `ex04-image-gallery` | `<table>` layout, `<img>` with `alt`/`title`, `<figcaption>` |
| 05 | `ex05-registration-form` | input types, `required`/`placeholder`/`autofocus`, `<output>` |
| 06 | `ex06-event-handling` | `onblur`, `onchange`, `onclick`, `ondblclick`, keyboard events |
| 07 | `ex07-video-invite` | `<video>`, `oncanplay`, `onbeforeunload` |
| 08 | `ex08-user-preferences` | `localStorage` / `sessionStorage`, restore + clear |
| 09 | `ex09-geolocation` | `getCurrentPosition`, error handling, high-accuracy options |
| 10 | `ex10-devtools-debugging` | Console logging, `console.table`, breakpoint practice |

## Notes

- **ex04** uses `placehold.co` placeholder images so the gallery renders without local assets.
- **ex07** streams a sample MP4 from w3schools; needs network access.
- **ex09** requires granting the browser location permission (and HTTPS or `localhost` in some browsers).
