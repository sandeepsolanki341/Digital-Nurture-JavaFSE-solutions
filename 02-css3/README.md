# Module 02 — CSS3

Theme: **Styling the Local Community Event Portal**. Eleven exercises covering CSS inclusion
methods, selectors, the box model, typography, responsive design, and debugging.

Each exercise folder has its own `index.html` and (where applicable) `styles.css`. Open the
`index.html` in a browser.

## Exercises

| # | Folder | Focus |
|---|--------|-------|
| 01 | `ex01-inclusion-methods` | inline vs. internal vs. external CSS |
| 02 | `ex02-syntax-comments` | clean rule structure, descriptive comments |
| 03 | `ex03-selectors` | universal, element, ID, class, grouping selectors |
| 04 | `ex04-color-background` | HEX, RGBA, background image + fallback, gradients |
| 05 | `ex05-typography` | Google Font, font properties, text transforms, spacing |
| 06 | `ex06-links-lists` | link pseudo-classes, list styling, nav without bullets |
| 07 | `ex07-table-styling` | borders, padding, zebra striping, `border-collapse` |
| 08 | `ex08-box-model` | margin/padding/border/outline, `visibility` vs `display` |
| 09 | `ex09-multi-column` | `column-count`, `column-gap`, `column-rule` |
| 10 | `ex10-media-queries` | responsive breakpoints, flex stacking, fluid sizing |
| 11 | `ex11-devtools-debug` | DevTools device toolbar, Elements & Network tabs |

## Notes

- **ex04** references a `placehold.co` background image; needs network access to display it
  (the fallback colour shows otherwise).
- **ex05** loads the Poppins font from Google Fonts.
- **ex08** intentionally demonstrates that `visibility: hidden` preserves layout space while
  `display: none` does not — inspect both in DevTools.
- **ex10 / ex11** are best viewed with the browser's responsive device toolbar enabled.
